== Description ==

### BlockArt Gutenberg Blocks – Best WordPress Gutenberg Block Plugin.

BlockArt brings you all the necessary Gutenberg blocks backed up by powerful features for seamless website building experience. Whether you're an experience designer or a beginner WordPress user, BlockArt is for you. We focused on making it simple and easy to use. Any user will adapt as they start designing. You can use BlockArt as a tool to bring your own design in to reality in these simple steps:

1. Add section
2. Choose columns for the section
3. Add blocks suitable for your design
4. Style it by making changes in settings

Or, simply choose pre-built template and import it. Then, you can just change the content as per your website.

### THE MOST USEFUL GUTENBERG BLOCKS AND COUNTING

* **Section:** Add Rows and Columns inside rows to create various layouts
* **Heading:** Create stylish title for each section with various markups from H1 to H6
* **Paragraph:** Customize typography and style paragraphs
* **Button:** Redirect users to your important pages or website by clicking buttons
* **Image:** Visual communication with your visitor via images
* **Spacing:** Give your designs room to breathe with white space

### KEY POINTS:

1. **Speed**
Making it lightweight and fast was the motto from the start, and we've achieved it
    * Reduced page size by limiting markup as much as possible
    * Only used-CSS are loaded in the page
    * Only used Google fonts are loaded
    * Blocks themselves are lightweight
    * Compatible with caching plugins

2. **Pre-built templates**
Import pre-build templates, sections and sites
    * Beautiful demo sites ready to import designed by professional UI/UX designs
    * You can import any sections/pages from any starter pack to the page
    * For better experience, related templates are listed while edition blocks
    * Live editing see actual design as you edit
3. **Copy and paste styles**
    * Click button in the block toolbar to copy style of any block
    * Works across any tabs of browser
    * Keyboard shortcuts available: Ctrl + C, Ctrl + Alt + V
4. **Responsive editing**
    * Responsive editing tools are provided in necessary controls of blocks like font size, line height, margin, padding, alignment, etc.
    * Show hide block in desktop, tablet, mobile
    * Preview in responsive mode while editing
5. **Dashboard widget area and customizer widget editor**
    * Works perfectly just like in page editor
    * Use BlockArt blocks in sidebar as well

### ONLY NECESSARY FEATURES, NO CLUTTER
* Unlimited layout possibilities
* Choice of units: px, em, %, rem
* Full width and contained section container
* Typography: font family font weight, line height, text transform, decoration, letter spacing
* Text colors
* Background color, background image and overlay
* Size and spacing: width, height, margin/padding, border/box shadow/border radius
* Horizontal and vertical alignment
